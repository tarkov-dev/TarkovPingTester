## 설명

타르코프 런처에서 사용하는 서버와 동일한 서버로 핑 테스트를 하는 프로그램입니다. 실행 즉시 핑 테스트를 진행하며, 10회 측정하여 평균값을 알려줍니다. 실제 게임 서버의 핑은 차이가 있을 수 있습니다.
